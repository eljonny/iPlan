To use the Send Email program
Enter the email address of the person to which you would like to send this email in the first box, labeled "To:"
Enter your email address, from which you are sending the message, in the second box, labeled "From:"
Enter the subject of the content of your message in the third box, labeled "Subject:"
Finally, enter the email message you would like to send to the recipient in the fourth box, labeled "Message:"
Functions of buttons
ButtonFunction
Send (flying message Image button) Clicking this button results in the text fields being checked for valid email addresses and empty subject line. If all texts fields appear to be valid, the program will attempt to send the message.
Cancel Clears all of the text fields back to default blank fields and asks if you would like to close the message.
Help Opens this help window
Restore If you clicked cancel by accident and chose to not close the message, clicking this button will restore your text fields to their content before you clicked cancel.
Context-Sensetive Help System
While focused on any text field, you can press the F1 key to get information about the text field.
